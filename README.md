# NUS Computing 50th Anniversary Bursary Fund (Static Site)

Minimal static marketing site for GitHub Pages deployment from the repository root.

## Files

- `index.html` - landing page content and structure
- `styles.css` - responsive styling (black/green/cream design language)
- `.nojekyll` - disables Jekyll processing on GitHub Pages
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow

## Content focus

The page centers on:

- Alumni passing opportunity forward
- SoC progress from a small department to global excellence
- Fundraising for students needing financial support
- Charity Golf & Dinner details and sponsorship tiers
